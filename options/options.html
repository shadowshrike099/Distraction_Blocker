How Emergency Unlock Works

  • When you start a focus session, you set a session-specific password
  • If you forget that password, use your master unlock code instead
  • You have a limited number of attempts to prevent brute-forcing
  • If max attempts are exceeded, wait for the session to end naturally
  • The master code is never stored in plain text - only a secure hash
